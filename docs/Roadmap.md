# Roadmap

Questo documento descrive le principali attività che verranno rilasciata nelle future versioni del software di ANSC.
Le date e la composizione degli interventi potrebbero subire variazioni.

- ID 327 Gestione annotazione di correzione per errore materiale (art.98) riferita ad altra annotazione
- ID 343 Trascrizione matrimonio dall'estero: gestione data di celebrazione parziale
- ID 347 Trascrizione del decreto prefettizio di cambio cognome/nome multi intestatario
- ID 321 Richiesta certificati plurilingua da sportello al cittadino
- ID 303 Recupero dal sistema di conservazione degli atti di morte ('scartati' digitalmente)
